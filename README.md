# 🚀 ShortForge AI

An AI-powered YouTube Shorts Generator built with FastAPI.

## Features

- 🤖 AI Script Generation (Ollama)
- 🎙 AI Voice Generation (Piper)
- 🎭 Multiple Script Styles
- 🌍 Multiple Languages
- ⏱ Video Duration Selection
- 📋 One-click Copy Script

## Tech Stack

- Python
- FastAPI
- Ollama
- Piper TTS
- HTML/CSS/JavaScript
- Git
