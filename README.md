# 🚀 ShortForge AI

An AI-powered short video generation platform built with **FastAPI**, **Ollama**, **Piper TTS**, **Pollinations AI**, and **MoviePy**.

Generate complete YouTube Shorts, Instagram Reels, and TikTok videos from a simple text prompt.

---

## ✨ Features

- 🤖 AI Script Generation (Ollama)
- 🎙 AI Voice Generation (Piper TTS)
- 🖼 AI Image Generation (Pollinations AI)
- 🎬 Automatic Video Creation (MoviePy)
- 📂 Project Dashboard
- 📜 Script Export
- 🎥 Video Preview
- 📥 Download MP4
- 🎧 Download Voice
- 📱 Vertical 1080×1920 Videos
- ⚡ FastAPI Web Interface

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yuvrajmishra863-star/ShortForge-AI.git
```

Move into the project

```bash
cd ShortForge-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run FastAPI

```bash
uvicorn main:app --reload
```

Open your browser

```
http://127.0.0.1:8000
```
